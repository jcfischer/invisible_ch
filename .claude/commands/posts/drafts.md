Find and list all draft blog posts in this Hugo site.

## Instructions

1. **Search for drafts** in `content/blog/` directory:
   - Look for files containing `draft: true` in frontmatter

2. **For each draft found**, display:
   - File path
   - Title
   - Date created
   - Category

3. **Format output** as a table:

| File | Title | Date | Category |
|------|-------|------|----------|
| ... | ... | ... | ... |

4. **Summary**:
   - Total number of drafts found
   - Suggest using `/posts:publish <filename>` to publish
